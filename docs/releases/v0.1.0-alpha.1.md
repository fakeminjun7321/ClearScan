# ClearScan v0.1.0-alpha.1

This is the first public alpha of ClearScan, an offline-first UIKit document
scanner for iPhone and iPad.

## Highlights

- Single-page and two-page book capture flows.
- Manual shutter plus recent-frame automatic-capture consensus and progress
  ring.
- A guarded fallback for edge-clipped, frame-filling book spreads with an
  off-center gutter.
- Perspective correction, page splitting, local OCR, enhancement, editing,
  folders, and real PDF/JPEG/ZIP export.
- Optional Google Drive upload and editable Google Docs OCR conversion.
- Companion React app and local document/page API.
- App icon, architecture, verification matrix, open-source scanner review,
  worktree collaboration guide, contribution guide, and CI.

## Verification

- Native unit tests: 90 passed, 0 failed.
- Backend tests: 5 passed, 0 failed.
- Companion browser tests: 3 passed, 0 failed.
- Mobile runtime tests: 8 passed, 0 failed.
- The sanitized app built and launched in an iOS Simulator.
- Offline replay of the supplied difficult book scan recovered 98.5% of the
  visible spread and estimated its off-center gutter at 65.9%.

## Honest alpha boundaries

This release does not claim current physical-device camera verification or a
completed live Google OAuth/Drive/Docs round trip. The simulator cannot test
camera hardware or shutter audibility. The original supplied study PDF and all
local OAuth/signing values are excluded from the repository.

[the verification matrix](https://github.com/fakeminjun7321/ClearScan/blob/main/docs/VERIFICATION_MATRIX.md)
